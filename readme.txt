Automatically migrate your WordPress blog from one URL to another URL.

== Description ==

Migrating WordPress has been very hard in the past. This plugin hopes to solve
that problem by autmatically copying your files and data to a new URL.

== Installation ==

1. Upload `wp-migration` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
